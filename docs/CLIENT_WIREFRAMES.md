# HarnessHub Client-Side Screen Specifications (화면 설계서)

본 문서는 올라랩(HarnessHub) 클라이언트(일반 사용자용)의 주요 화면 레이아웃과 데이터 바인딩 요소를 마크다운 구조도를 통해 명세합니다.

---

## 1. 메인 홈 (Home Dashboard)
**경로:** `/` (Root)
**목적:** 사용자가 서비스에 진입했을 때 플랫폼의 핵심 콘텐츠(트렌딩 툴, 베스트 커뮤니티 글)를 가장 먼저 노출하여 체류 시간을 늘립니다.

### 📐 레이아웃 구조

```text
+-------------------------------------------------------------+
| [GNB] 로고(좌) | 도구 | 커뮤니티 | 실험실 |         [로그인/가입] |
+-------------------------------------------------------------+
| [상단 배너] 📣 [공지] 26년 5월 오프라인 실험실 모임 신청 안내        |
+-------------------------------------------------------------+
|                                                             |
|                      [Hero Section]                         |
|         일상을 더 창의롭게 만드는 AI 커뮤니티 (타이틀)             |
|           [도구 탐색] [AI 실험실] [글 쓰기] (버튼 3종)          |
|         [ 🔍 어떤 AI를 함께 써보고 싶으신가요? (검색바) ]           |
|                                                             |
+-------------------------------------------------------------+
| [Main Column] (좌측 70%)         | [Sidebar] (우측 30%)     |
|                                |                            |
| 1. 오늘의 인기 트렌드 (모바일 대응)  | 1. 실시간 인기 도구 랭킹       |
|  - 🔥 오늘의 인기 도구 (Top 3)    |  - 1위 Cursor             |
|  - 🏆 이번주 베스트 글 (Top 3)    |  - 2위 Midjourney         |
|                                |                            |
| 2. AI 도구 리스트 (Grid)          | 2. AI 실험실 참여 CTA 카드    |
|  - [카드1] [카드2] [카드3]       |                            |
|  - [카드4] [카드5] [카드6]       | 3. 인기 커뮤니티 글 리스트     |
|                                |                            |
|  [ 도구 더보기 버튼 ]             | 4. 인기 카테고리 태그 모음     |
+-------------------------------------------------------------+
| [Footer] 카피라이트 및 정책 링크                                |
+-------------------------------------------------------------+
```

### 🔗 데이터 바인딩
* `Hero Section` 하단 마키(Marquee) 애니메이션: `GET /products?limit=40` 의 아이콘 배열
* `오늘의 인기 트렌드`: `GET /products/ranking` 상위 3개 및 `GET /posts/ranking` 상위 3개

## 2. AI 도구 탐색 (Products Directory)
**경로:** `/products`, `/products/[id]`
**목적:** 101개의 AI 도구를 카테고리별로 쉽게 찾고, 상세 정보와 평점을 확인합니다.

### 📐 레이아웃 구조 (상세 페이지)

```text
+-------------------------------------------------------------+
| [GNB] ...                                                   |
+-------------------------------------------------------------+
| [ Product Header Section ]                                     |
|  +----+  [도구 이름 (ex. Cursor)]  [공식 사이트 방문 버튼]        |
|  |아이|  평점: ⭐️ 5.0 | 프라이싱: Freemium                     |
|  | 콘 |  카테고리: 개발 | 랭킹: 1위                             |
|  +----+                                                     |
+-------------------------------------------------------------+
| [ Main Content ]                                            |
|  - 도구 한줄 소개 (Short Description)                         |
|  - 상세 설명 (Full Description)                               |
|                                                             |
| [ 스크린샷 / 썸네일 갤러리 ]                                     |
|  +------------------+  +------------------+                 |
|  | 이미지 1          |  | 이미지 2          |                 |
|  +------------------+  +------------------+                 |
+-------------------------------------------------------------+
```

## 3. 커뮤니티 (Community)
**경로:** `/community`, `/community/write`
**목적:** 유저들끼리 자유롭게 질문하고, 후기를 남기며 소통하는 공간입니다.

### 📐 레이아웃 구조 (피드 목록)

```text
+-------------------------------------------------------------+
| [ 카테고리 탭 ] 전체 | 자유게시판 | 정보공유 | 질문/답변              |
|                                                             |
| [글쓰기 버튼] (우측 상단)                                        |
+-------------------------------------------------------------+
| [ 커뮤니티 글 아이템 1 ]                                        |
|  제목: Cursor 업데이트 이후 해외 논문 번역 워크플로우 변화           |
|  작성자: 프롬프트장인 | 시간: 2시간 전 | 조회수: 142 | 💬 12       |
+-------------------------------------------------------------+
| [ 커뮤니티 글 아이템 2 ]                                        |
|  제목: 요즘 영상 생성은 Sora가 답인가요?                          |
|  작성자: 퇴근좀시켜줘 | 시간: 5시간 전 | 조회수: 89  | 💬 4        |
+-------------------------------------------------------------+
```

### 💡 주요 상호작용 (Interaction)
* **글쓰기:** 20자 최소 글자 수 제한이 **해제**되어 짧은 핑퐁 대화 텍스트도 입력 및 제출 가능
* **댓글:** 작성자 아이디(UUID) 대신 '한국어 닉네임' 우선 노출

## 4. AI 실험실 (Market)
**경로:** `/market`
**목적:** 2시간 분량의 오프라인/온라인 실습 모임에 대한 안내 및 큐레이션 정보를 제공합니다.

### 📐 레이아웃 구조

```text
+-------------------------------------------------------------+
| [ Market Header ]                                             |
| 🧪 오프라인 모임: 직접 만들며 배우는 AI 실험실                       |
+-------------------------------------------------------------+
| [ 카드 1: 나만의 고품질 AI 팟캐스트 제작하기 ]                      |
|  - 스택: NotebookLM, ElevenLabs                             |
|  - 메트릭: 소요 시간 2시간                                      |
|  - 좋아요: ❤️ 45                                             |
+-------------------------------------------------------------+
| [ 카드 2: Cursor와 Claude로 2시간 만에 웹 게임 배포하기 ]           |
|  - 스택: Cursor, Claude 3.5 Sonnet                          |
|  - 메트릭: 소요 시간 2시간                                      |
|  - 좋아요: ❤️ 82                                             |
+-------------------------------------------------------------+
```
