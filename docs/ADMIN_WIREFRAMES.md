# HarnessHub Admin-Side Screen Specifications (화면 설계서)

본 문서는 올라랩(HarnessHub) 관리자용 백오피스 앱의 주요 화면 레이아웃과 데이터를 처리하는 명세를 기술합니다.

---

## 1. 대시보드 (Dashboard)
**경로:** `/` 
**목적:** 관리자가 플랫폼의 전체적인 볼륨과 트래픽 현황을 한눈에 파악합니다.

### 📐 레이아웃 구조

```text
+-------------------------------------------------------------+
| [Sidebar]       | [Header] 관리자님, 환영합니다.                 |
| - 대시보드        +---------------------------------------------+
| - 도구 관리       | [ 요약 위젯 영역 ]                            |
| - 커뮤니티 관리    |  [총 등록 도구]  [대기 중인 도구]  [금주 게시글]  |
| - 유저 관리       |      101건           3건             45건     |
|                 |                                             |
|                 | [ 최근 유입 트렌드 그래프 영역 ]                  |
|                 |  (Line Chart)                               |
+-------------------------------------------------------------+
```

---

## 2. AI 도구 관리 및 심사 (Products List & Pending)
**경로:** `/products`
**목적:** 신규 등록된 도구를 심사하여 라이브 뷰에 노출시키거나 기존 도구의 정보를 수정합니다.

### 📐 레이아웃 구조

```text
+-------------------------------------------------------------+
| [ 도구 관리 ]                                                  |
| 탭: [전체 도구 (101)] | [승인 대기 (3)]                           |
|                                                             |
| [필터/검색바] 검색어 입력...                                      |
+-------------------------------------------------------------+
| 데이터 테이블                                                  |
| ID | 썸네일 | 도구명   | 카테고리  | 상태      | 액션             |
| 1  | [img] | Cursor  | 개발     | PUBLISHED | [수정] [삭제]     |
| 2  | [img] | 미확인앱 | 마케팅   | PENDING   | [심사하기]        |
+-------------------------------------------------------------+
```

### 💡 주요 상호작용: 도구 상세 심사 모달 (ContentDetailModal)
위 테이블에서 **[심사하기]** 버튼 클릭 시 화면 중앙에 모달 팝업이 노출됩니다.

```text
+------------------------------------------------+
| 도구 심사: '미확인앱'                              |
+------------------------------------------------+
| 썸네일: [이미지 미리보기]                           |
| 설명: 마케팅 문구를 자동으로 써주는 어쩌구...           |
| 제출자: user123@harness.kr                          |
|                                                |
|            [반려하기(Reject)]  [승인하기(Approve)]|
+------------------------------------------------+
```
- **데이터 바인딩:** `approve` 시 `PUT /api/admin/products/:id/approve` 호출.

---

## 3. 커뮤니티 모니터링 (Community Management)
**경로:** `/community`
**목적:** 유저 간의 분쟁, 스팸 홍보성 글 등을 모니터링하고 강제 삭제 조치합니다.

### 📐 레이아웃 구조

```text
+-------------------------------------------------------------+
| [ 커뮤니티 글 관리 ]                                             |
+-------------------------------------------------------------+
| 데이터 테이블                                                  |
| 게시글 ID | 제목                 | 작성자 닉네임 | 작성일 | 액션   |
| 104    | 안녕하세요 가입했습니다! | 퇴근좀시켜줘 | 오늘   | [삭제] |
| 105    | 불법 도박 사이트 홍보   | 봇계정22   | 어제   | [삭제] |
+-------------------------------------------------------------+
```
